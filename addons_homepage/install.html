Additional software need more?

The Eclipse Foundation has very strict rules when it comes to Licenses so for example LGPL software can not be shipped from Eclipse.org servers. If you want company has less strict rules we provide you additional software you can install in your IDE or make use of at runtime.

In case you downloaded an all-in-one download from above you already have the IDE-addons installed by default. For those who used the Eclipse-Update-Site to install e(fx)clipse you can post install the addons from http://downloads.efxclipse.bestsolution.at/update-sites/releases/latest
